# 🚀 초보자를 위한 Google Calendar 봇 간단 시작 가이드

## 🎯 **이 가이드의 목적**

**"복잡한 설명은 나중에, 일단 작동시켜보자!"**

코덱스가 지적한 대로 초기 가이드가 너무 복잡했습니다. 
이 가이드는 **핵심만 담은 최소한의 설정**으로 빠르게 체험해볼 수 있도록 구성했습니다.

---

## ⏰ **시간 투자 예상**

### 🟢 **최소 목표**: 30분 내로 첫 테스트 완료
- Google 설정: 15분
- 서버 실행: 5분  
- 메신저봇 테스트: 10분

### 🟡 **현실적 목표**: 1시간 내로 안정적 작동
- 문제 해결 시간 포함

---

## 🔥 **1단계: Google 설정 (15분)**

### ✅ **딱 이것만 하세요**

1. **Google Cloud Console 접속**
   ```
   https://console.cloud.google.com
   → 프로젝트 만들기 → 이름: "캘린더봇테스트"
   ```

2. **API 활성화** (클릭 2번)
   ```
   좌측 메뉴 → API 및 서비스 → 라이브러리
   → "Calendar" 검색 → Google Calendar API → 사용 설정
   ```

3. **인증 파일 다운로드** (클릭 5번)
   ```
   좌측 메뉴 → API 및 서비스 → 사용자 인증 정보  
   → + 사용자 인증 정보 만들기 → OAuth 클라이언트 ID
   → OAuth 동의 화면 구성 (필요시) → 외부 → 만들기
   → 앱 이름: "테스트봇" → 저장
   → OAuth 클라이언트 ID → 애플리케이션 유형: 데스크톱 앱 → 만들기
   → JSON 다운로드 (파일명: credentials.json)
   ```

### 🛑 **여기서 막히면**
- "앱 확인되지 않음" 오류 → **무시하고 진행**
- OAuth 동의 화면 → **최소한만 입력**
- 복잡한 설정 → **나중에 해도 됨**

---

## 🐍 **2단계: 서버 실행 (5분)**

### ✅ **터미널에서 순서대로 실행**

```bash
# 1. 프로젝트 폴더로 이동
cd examples/GoogleCalendarBot

# 2. credentials.json 파일을 이 폴더에 복사

# 3. 가상환경 생성 (처음에만)
python3 -m venv venv

# 4. 가상환경 활성화
source venv/bin/activate  # Mac/Linux
# venv\Scripts\activate   # Windows

# 5. 패키지 설치 (처음에만)
pip install -r requirements.txt

# 6. 서버 실행
python fastapi_server.py
```

### 🟢 **성공 확인**
```
브라우저가 자동으로 열림 → Google 로그인 → 권한 허용
터미널에 "서버 시작됨" 메시지 출력
```

### 🛑 **여기서 막히면**
- Python 없으면 → python.org에서 설치
- 패키지 오류 → `pip install --upgrade pip` 후 재시도
- 브라우저 안열림 → 수동으로 출력된 URL 클릭

## 📱 **3단계: 메신저봇 테스트 (10분)**

### ✅ **코드 수정**

`GoogleCalendarBot.js` 파일에서 **딱 2줄만 수정**:

```javascript
var CONFIG = {
    SERVER_URL: "http://localhost:9000",      // 로컬 테스트용
    TARGET_ROOMS: ["나와의채팅"]              // 나와의채팅으로 테스트
};
```

### ✅ **메신저봇 설정**
1. **메신저봇 R 앱** 설치 및 실행
2. 코드 복사 → 붙여넣기 → **컴파일**  
3. 봇 **실행**

### ✅ **테스트 명령어**
카카오톡 "나와의채팅"에서:
```
서버 상태     → 연결 확인
일정         → 오늘 일정 보기  
빈시간 오늘   → 빈 시간 확인
```

### 🟢 **성공하면**
```
✅ Google Calendar 서버가 정상 작동 중입니다.
📅 오늘 일정이 없습니다.
🕐 2024-01-15 (월요일) 빈 시간:
⭕ 09:00 - 19:00
```

## 🚨 **문제 해결 (간단 버전)**

### ❌ **"서버 연결 실패"**
```
해결: python fastapi_server.py가 실행 중인지 확인
     브라우저에서 http://localhost:9000 접속해보기
```

### ❌ **"OAuth 오류"**  
```
해결: credentials.json 파일이 올바른 위치에 있는지 확인
     브라우저에서 Google 인증 다시 시도
```

### ❌ **"메신저봇 컴파일 실패"**
```
해결: ES5 문법 사용 (const → var, 화살표 함수 금지)
     코드 복사 시 모든 내용이 포함되었는지 확인
```

### ❌ **"일정이 안보여요"**
```  
해결: Google 계정에 실제 캘린더 일정이 있는지 확인
     다른 캘린더 앱에서 일정 하나 추가해보기
```
